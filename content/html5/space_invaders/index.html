Objective:

  • The objective of the game is to kill all the enemies, mothership is a bonus.
  • The objects are generated through bitmaps.
  • The game will be over when either the player looses all the lives or all the aliens get killed.


Logic Design and Implementation:

  • Animation of the objects is done by changing bitmaps.
  • Concept of object Pooling is used in firing bullets.
  • Only the last row aliens of the columns shoot bullets.
